# Beauty Secrets 

Beauty Secrets is a crud app that allows users to upload skin care products that they enjoy using. The user is able to make notes about each product that they have used. 


## Technologies Used
- Express
- MongoDB
- CSS3
- HTML5
- Morgan 
- Method-Overide
- Mongoose
- Ejs
