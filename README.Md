# Beauty Secrets 

Beauty Secrets is a crud app that allows users to upload skincare products that they enjoy using. The user can make notes about each product that they have used. 


## Technologies Used
- Node.js
- Express
- MongoDB
- Mongoose
- CSS3
- HTML5
- Morgan 
- EJS
- Google Fonts
